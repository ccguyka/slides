JGiven

How do you arrange your tests?

Put everything into the test method


@Test
public void adult_gets_a_beer_in_a_german_bar() {
	//GIVEN
	User user = new User();
	user.setAge(22);
	Bar bar = new Bar();
	bar.setCountry(GERMANY);

	//WHEN
	user.ordersBeerIn(bar);

	//THEN
	assertThat(user).hasBeer();
}
Could get messy with heavy objects. Solution: Builders.
A lot of boilerplate code.

Move everything into new methods


@Test
public void adult_gets_a_beer_in_a_german_bar() {
	givenAdultAndGermanBar();

	whenAdultOrdersBeer();

	thenAdultHasBeer();
}
Test class will contain a lot of methods.
